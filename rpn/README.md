rpn
===========
A simple calculator implements via [Reverse Polish Nation](https://en.wikipedia.org/wiki/Reverse_Polish_notation)

## Infix Notation
```
(4 - 2) * 5
```
## Reverse Polish Notation
```
4 2 - 5 *
```
## How to play
run app, then input:
```
4
2
-
5
*
=
```
'clear': to clear the calculated result

'quit': to quit the app
